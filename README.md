🎮 Hangman Pro (Python)

A simple but well-structured implementation of the classic Hangman game built using Python.
This project includes both GUI (Tkinter) and Web (Flask) versions based on a shared core logic.

🚀 Features
Object-Oriented Programming (OOP) design
Shared core game logic (modular structure)
GUI version using Tkinter
Web version using Flask
Category-based word selection
Input validation (prevents invalid guesses)
Extendable architecture for future improvements
🧠 Game Logic

The game randomly selects a word from predefined categories and the player must guess it letter by letter before running out of lives.

🖥️ How to Run (GUI Version)
python gui.py
🌐 How to Run (Web Version)
python web.py

Then open in browser:

http://127.0.0.1:5000/
