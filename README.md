# 🎮 Hangman Pro - Full Stack Web Game

A modern full-stack Hangman game built with Python (Flask) and deployed on Railway.

---

## 🚀 Live Demo
👉 https://hangman-pro-sd.up.railway.app

---

## 📌 Features

- 🎯 Category-based gameplay (Animals / Programming / Countries)
- ⚡ Difficulty levels (Easy / Medium / Hard)
- 🎮 Win / Lose system
- 🔁 New Game & Reset functionality
- 📊 Game state tracking (lives, guessed letters)
- 🎨 Responsive UI using Bootstrap
- ☁️ Cloud deployment (Railway)

---

## 🛠️ Tech Stack

- Python 3
- Flask
- HTML / CSS
- Bootstrap 5
- Git & GitHub
- Railway (Deployment)
